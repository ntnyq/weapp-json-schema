> JSON schema for weapp.

## Usage

### Project config

```jsonc
// Add line bellow to your `project.config.json`
{
  "$schema": "https://jsonschema.ntnyq.com/project.schema.json"
}
```

## Schemas

- [project.schema.json](./project.schema.json) Schema for `project.config.json`
